# PARTNER

This repository contains the dataset and code for our paper titled [PARTNER: A Persuasive Mental Health and Legal Counselling Dialogue System for Women and Children Crime Victims](https://www.ijcai.org/proceedings/2023/0686.pdf)

## Code
The folder PARTNER contains the code for the proposed PARTNER architecture.
